### üíæ Installation

1.  Clone the repository:
    ```sh
    git clone [https://github.com/Ayushx404/career-path-finder-and-advisor.git](https://github.com/Ayushx404/career-path-finder-and-advisor.git)
    ```
2.  Navigate to the project directory:
    ```sh
    cd career-path-finder-and-advisor
    ```
3.  Install backend dependencies:
    ```sh
    cd backend
    npm install
    ```
4.  Install frontend dependencies:
    ```sh
    cd ../frontend
    npm install
    ```
5.  Set up environment variables:
    *(**Note to author:** Explain how to set up `.env` files. Create a `.env.example` file in both the `backend` and `frontend` directories for users to copy.)*
    * Create a `.env` file in the `/backend` directory and add your environment variables (e.g., `DATABASE_URL`, `JWT_SECRET`).
    * Create a `.env` file in the `/frontend` directory (if needed) for variables like `REACT_APP_API_URL`.
